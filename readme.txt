== Description ==

썬더포트는 워드프레스.org 플러그인의 서포트 페이지를 쉽게 검색하기 위해서 만들었습니다.

썬더포트를 이용해서 플러그인의 서포트 페이지를 검색하는 시간을 줄여보세요.

각 플러그인 마다 "번개" 모양의 아이콘을 클릭하면 서포트 데이터 파싱이 시작됩니다.

그리고 모든 플러그인을 한번에 검색할 수 있습니다.

(플러그인 이용자가 많아지면, 플러그인별 질문을 쉽게 올릴 수 있는 기능도 연동할 예정입니다.)

Thunder Port is a plugin for quick search in wordpress plugin's support.

Save time to search question in your WordPress plugin's support.

Click "Thunder" icons for each plugin.

Search all plugins at once.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/thunder-port` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
